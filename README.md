# 🐧 Linux Administration Course

Интерактивный курс для изучения Linux с нуля до уверенного администратора.

## 🎯 Для кого этот курс?

- **Новички**, которые хотят стать Linux-админами
- **Разработчики**, которым нужно администрировать серверы
- **Студенты IT-специальностей**
- **Системные администраторы Windows**, переходящие на Linux

## 📚 Содержание

1. **Файлы и навигация** — дерево каталогов, ls, cd, cp, rm, find, grep
2. **Права доступа** — chmod, chown, SUID, SGID, umask
3. **Процессы** — ps, top, htop, kill, zombie-процессы
4. **Пакеты и сервисы** — apt, systemctl, репозитории
5. **Демоны и systemd** — unit-файлы, таймеры, journalctl
6. **Безопасность** — SSH-ключи, fail2ban, UFW
7. **Траблшутинг** — методология диагностики, dmesg, /proc
8. **Логи** — syslog, journalctl, jq, logrotate
9. **Сеть** — DNS, SSH-туннели, tcpdump, фаервол
10. **Bash-скрипты** — автоматизация, Git, cron
11. **Мониторинг** — Prometheus, Grafana, алерты в Telegram
12. **Хранение** — LVM, fstab, SMART
13. **Docker** — контейнеры, Compose, best practices

## ✨ Особенности

- ✅ **Troubleshooting в каждом модуле** — что делать, когда не работает
- ✅ **Реальные production-сценарии** — не абстрактные примеры
- ✅ **Задачи со звёздочкой** — для продвинутых студентов
- ✅ **Работает офлайн** — не нужен интернет
- ✅ **Прогресс сохраняется** — можно вернуться в любой момент
- ✅ **Тёмная/светлая тема** — на выбор
